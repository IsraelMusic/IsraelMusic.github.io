# IsileliMusic Website

Static GitHub Pages site for IsileliMusic LLC.

## Files

- `index.html` — homepage
- `assets/styles.css` — design system + responsive layout
- `assets/script.js` — mobile nav + dynamic year

## Publish on GitHub Pages

1. Create a GitHub repo, e.g. `isilelimusic.github.io` or `isilelimusic-site`.
2. Upload these files to the repo root.
3. In GitHub: Settings → Pages → Deploy from branch → `main` / root.
4. Add your custom domain later if needed.

## Customize next

Replace placeholder social links and email in `index.html`.
